# Title

Deep Learning-Based Detection of Mitral Regurgitation from Echocardiography Images

## Team members

Junpyo Lee (JunpyoLee-985)

## Project description

This project aims to develop a deep learning model to detect Mitral Regurgitation (MR) from echocardiography images. Mitral Regurgitation is a common heart valve disorder in which the mitral valve does not close properly, causing blood to flow backward into the left atrium. Early and accurate detection is important for clinical decision-making and patient management.

The dataset will consist of echocardiography images (such as color Doppler or ultrasound frames) collected from publicly available medical imaging datasets or curated image sources. The dataset will be organized into multiple classes, such as normal (no MR), mild MR, and severe MR, depending on availability.

The goal of this project is to train a neural network model (e.g., a convolutional neural network) to classify images based on the presence and severity of Mitral Regurgitation. Different model architectures and parameters (such as learning rate, number of layers, and image resolution) will be explored to evaluate performance.

The model will be trained and evaluated using training and test datasets, and performance metrics such as accuracy and loss will be analyzed. The final objective is to assess whether a simple deep learning approach can reliably distinguish between normal and abnormal cardiac function related to Mitral Regurgitation.
